# Docs Magazine

This folder is the *companion* to the technical [README](../../README.md).

The README answers **how**. The magazine answers **why**.

It is a deliberate extra deliverable — not required by the assignment, written because the topic of "what should modern tooling do for a 1968 teaching language?" deserves more than a bullet list. The audience is whoever wonders, after the code review is done, what this project was trying to *be*.

Read in any order:

- [`issue-01.md`](issue-01.md) — *Issue №01: A turtle, a tree, and a typed editor.* The full magazine: an essay on LOGO and why it still teaches, a deep look at the inlay-hint feature as pedagogy, a sketch of the concurrency contract in plain terms, and the editorial decisions about what was cut.
